  1. TRB sequencing

    1. T cell receptor beta chain gene rearrangements were bulk sequenced from gDNA using similar methods to IGH sequencing.  See the IGH protocol here.
    2. Further details, including PCR primers, can be found in PMID 30111633 and Meng et al., 2021 .
    3. For data analysis, we recommend using the ImmuneDB pipeline (PMID 30298069), with database installation instructions available on GitHub.
      4. Note: A detailed step-by-step protocol for all of the wet and dry bench methods is expected to be available in Methods in Molecular Biology later in 2022.
