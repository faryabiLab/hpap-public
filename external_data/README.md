### This directory includes some external data for HPAP.

* `dna_methylome_bigwigs.json`: originally created by `Elisabetta Manduchi` on
  12/09/2025.

* `create_dna_methylome_json.py`: a Pythojn script that reads
  `dna_methylome_bigwigs.json` to generate `dna_methylome.json` file.

  Note: `Python 3.9+` is required to run this script.

* `dna_methylome.json`: JSON file that represents the fake DNA methylome data.
  It will be used by both front-end and back-end.
